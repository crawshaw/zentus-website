Fleet Graphical Debugger

The Fleet Graphical Debugger, fgd, helps you visualise the execution of your Fleet programs.

Fleet is a clockless, non-Von Neumann processor architecture. Instead of a program counter and a linear set of instructions, the processor is made up of ships, each with multiple docks. You program a Fleet by programming the docks. Each dock communicates with its ship and other
